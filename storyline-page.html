Guidelines
Visual Design

A consistent visual approach ensures a seamless learner experience and brand alignment. All Storyline modules should follow the core design system.

Canvas & Layout
1920x1080px (16:9) 12 column

Typography
Headings: Century Gothic (Bold)
Body: Open Sans (Regular)

Colours
Use the organisation’s primary and secondary brand colours.
Maintain WCAG 2.1 AA contrast standards.

Layout
Align content using internal 12-column or proportional layouts.
Maintain consistent spacing between elements.
Use Storyline’s built-in guides for alignment.

Imagery & Icons
Optimise images for web (JPG for photos, PNG for UI/transparent, SVG for icons).
All icons should be consistent and sourced from the approved design library.

Navigation
Use custom navigation buttons.
Default positions: Back (left), Next (right), Submit (centre or near form fields).
Hide the default Storyline player unless specifically required.

Component Reuse
Create reusable slide templates (e.g., objectives, knowledge checks, scenarios).
Maintain consistent placement of elements across slides.

Technical Implementation

Canvas & Layout Fixed responsive layout — adapt visual elements to the stacking format. Use section dividers and padding to structure content correctly.

Typography Limited by Rise defaults. Select closest matching fonts to brand guidelines. Headings: bold and concise; Body: minimal and readable.

Colour Palette Set primary and accent colours in Theme > Branding. Match brand colours as closely as possible; use high contrast.

Imagery & Icons Use wide, high-quality banner images (1920x1080px recommended). Use consistent icon style, aligned to brand (SVG or PNG with transparency).

Navigation Leverage Rise’s natural scroll experience. Use sidebar or navigation menu to structure topics clearly.

Content Blocks Use labelled block groups for quick reuse (e.g., Accordion_Legal, Tabs_Scenarios). Group content logically within lessons to keep the scroll manageable.

Accessibility

Alt Text Fixed responsive layout — adapt visual elements to the stacking format. Use section dividers and padding to structure content correctly.

Contrast Limited by Rise defaults. Select closest matching fonts to brand guidelines. Headings: bold and concise; Body: minimal and readable.

Keyboard & Screen ReadersRise blocks are largely accessible out-of-the-box. Test in your LMS.

Closed CaptionsUpload transcript for embedded videos.

StructureEnsure headings are used logically (H1, H2, etc. based on block structure).

Publishing

TestingUpload to SCORM Cloud or LMS staging area
Check tracking, progression, and media compatibility
